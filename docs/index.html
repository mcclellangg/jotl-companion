JOTL Companion

JOTL Companion is a set of tools for Gloomhaven: Jaws of the Lion players to track character progress, encounters, gold, XP, items, and notes. All tools are mobile-friendly and support data export/import via JSON.

Tools

  • Scenario Log - Track XP, gold, and notes for each scenario and event.

Character Sheets

Manage stats, perks, items,and notes for your character.
